# シンプルテトリス

## 概要

シンプルで遊びやすいテトリスゲームです。Vanilla JavaScript を使用して開発しました。

デプロイ：
https://masvc.github.io/Tetris/

## 主な機能

- 基本的なテトリスのゲームプレイ
- 7 種類のテトリミノ（ブロック）の落下
- ライン消去機能
- スコア表示
- ゲームオーバー判定
- ネクストブロックの表示
- ハイスコアの保存

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript

## 操作方法

- 左矢印キー: ブロックを左に移動
- 右矢印キー: ブロックを右に移動
- 下矢印キー: ブロックを下に移動（ソフトドロップ）
- 上矢印キー: ブロックを回転
- スペースキー: ハードドロップ（即座に落下）
